# FlowerShop

**Полнофункциональное WPF приложение для управления цветочным магазином**

Современное настольное приложение на C# для эффективного управления работой цветочного магазина с поддержкой веб-интеграций.

## Основные возможности

### Для клиентов:
- Просмотр каталога цветов и букетов
- Система управления корзиной
- Оформление и отслеживание заказов
- Оставление отзывов и рейтингов
- Управление профилем и историей покупок

### Для администраторов:
- Полное управление каталогом товаров
- Управление клиентской базой
- Управление заказами и доставками
- Администрирование категорий
- Загрузка и управление изображениями

## Технологии

- **Frontend**: WPF (XAML)
- **Backend**: C# с Entity Framework Core
- **БД**: PostgreSQL
- **Безопасность**: BCrypt
- **Архитектура**: MVVM

## Функции безопасности

- Хеширование паролей BCrypt
- Разделение прав доступа (Admin/Customer)
- Валидация входных данных
- Управление сеансами пользователя
